# LectureSync: Automated Quiz & Study Note Generator

LectureSync is an AI-powered system that automatically converts lecture videos into concise study notes and interactive quizzes.

It eliminates the manual effort teachers spend on summarization and assessment creation.

---

## Problem

Teachers spend significant time:
- Watching recorded lectures
- Extracting key concepts
- Writing quizzes and summaries manually

This process is repetitive, time-consuming, and inefficient.

---

## Solution

LectureSync automates the entire workflow using AI.

### Input
- Uploaded lecture video (MP4)
- YouTube lecture link

### Output
- Downloadable summarized PDF study notes
- Automatically generated 10-question quiz based on the lecture content

---

## Key Features

### 1. AI Transcription
Lecture audio is converted to accurate text using speech-to-text models.

### 2. Concept Extraction
Key topics and learning objectives are identified from the transcript.

### 3. Study Notes Generation
A structured, easy-to-read summary is generated and exported as a PDF.

### 4. Quiz Generation
A content-specific quiz is generated to assess understanding of the lecture.

---

## Target Users
- Teachers and professors
- Online educators
- Students revising recorded lectures
